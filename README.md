<h1 align="center">ant-back</h1>
  
![image](https://img.shields.io/badge/license-MIT-green)

基于ant-design-pro实现的通用后台，后端使用Koa实现。目前仍在开发中。
<br>
本项目的开发目的在于快速开发一个网站后台。

## 功能
- 权限管理（按钮级）
- 菜单管理
- 用户管理
- 文章管理
